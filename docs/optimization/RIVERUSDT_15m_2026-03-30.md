## Key Findings

### 1. Trailing Stop is the #1 Driver
- Trailing stop mult has the **biggest impact** on profitability
- Default (10) → Optimized (4): **+795% PnL improvement**
- Tighter trailing locks profits faster, dramatically improving win rate

### 2. KNN Parameters Have Zero Effect
- `knn.neighbors` (2-16) and `knn.lookback` (20-200) produce **identical results**
- The ribbon filter (threshold=5) dominates signal generation
- KNN classifier is effectively a no-op with current filter configuration

### 3. ADX and CVD Filters Are Redundant
- `adx_threshold` (5/8/12) — no effect
- `cvd_threshold` (0.3/0.5/0.9) — no effect
- Only `ribbon.threshold` materially affects signals

### 4. Order Size Controls Risk/Reward Tradeoff
- **os=50:** 331% PnL, **25.2% DD** (safest)
- **os=75:** 985% PnL, 42.6% DD (balanced)
- **os=100:** 1621% PnL, 52.0% DD (aggressive)

### 5. Stop Loss Sweet Spot
- SL=0.5 — loses money (too tight)
- SL=1.0 — marginal
- **SL=1.5-2.0 — optimal** (2.0 slightly better with tight trailing)
- SL>2.5 — increased DD, lower PnL
