# /optimize-analyze — Анализ результатов оптимизации

Читает последний отчёт из `docs/optimization/` и выводит ключевые выводы.

## Шаги

1. Найти последний файл в `docs/optimization/`
2. Прочитать отчёт
3. Вывести:
   - TOP-3 конфигурации с метриками
   - Какие параметры больше всего влияют на результат
   - Рекомендация: что менять дальше
   - Сравнение с текущим активным конфигом
