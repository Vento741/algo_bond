# /optimize-apply [CONFIG_ID] — Применить оптимальный конфиг

Обновляет активного бота на использование оптимизированного конфига стратегии.

## Шаги

1. Если CONFIG_ID не указан — найти последний оптимизированный конфиг (по имени "*Optimized*")
2. Найти активного бота пользователя (status=running)
3. Остановить бота
4. Создать нового бота с оптимизированным конфигом
5. Запустить нового бота
6. Подтвердить переключение
